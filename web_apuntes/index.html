variables

  • ¿Qué es var?

    Es un tipo de variable que no lleva tipo de dato, se escribe varNombreVariable;

    Ejemplo 1 
    
const nombreUsuario = "Carlos";  // No va a cambiar
let puntos = 0;  // Va a ir aumentando en el juego

puntos = puntos + 10;  // ✅ Perfecto con let

    Ejemplo 2 
    
if (true) {
var nombre = "Suscribete";

console.log(nombre);
                }

    Está bien ✅. En la consola, al ejecutar el código se imprime Suscribete.


  • ¿Qué es let?

    Se declara de la misma forma que el var. Sin embargo, se usa solo cuando que usar una variable en unicamente una estuctura, bloque o una funcion en particular, declara que solo se puede usar ahí y no en otro lugar.

    Ejemplo 1 
    
let contador = 0;
contador = 1; // ✅ Funciona bien
contador = 2;  // ✅ Sigue funcionando

let edad = 20;
edad = edad + 1; // ✅ Perfecto para ir cambiando valores

    Ejemplo 2 
    
if (true){
let nombre = "Suscribete";
}

console.log(nombre);

    Esto no se puede hacer, da error ❌. No existe nombre afuera, se encuentra adentro del bloque if solamente, esto ocurre porque es un let, un tipo de dato que solo funciona en un lugar especifico, no en todas partes.


  • ¿Qué es const?

    Es una variable que a lo igual que el let, se puede ocupar solo en un solo lugar en particular, lo unico diferente es que tiene un valor constante... es decir, que no puede cambiar. Si le vamos un valor a alguien, por ejemplo Lucia, Lucia no podrá dejar de llamarse Lucia, jamás.

    Ejemplo 1 
    
const nombre = "María";
const edad = 25;
const PI = 3.14159;
const precio = 100;
precio = 200; // ❌ Error!
    Ejemplo 2 
    
const email = "nombre@mail.com";
email = "otro_mail@mail.com"

    Esto no se puede hacer ❌. Porque email ya tiene un nombre asignado , a email no le puedo dar otro valor, es único.
