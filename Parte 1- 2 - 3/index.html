variables

  • ¿Qué es el var?

    Es un tipo de variable que no lleva tipo de dato, se escribe varnombreVariable;

    Ejemplo 

    
                const nombreUsuario = "Carlos";  // No va a cambiar
    
                let puntos = 0;  // Va a ir aumentando en el juego
    
                puntos = puntos + 10;  // ✅ Perfecto con let

  • ¿Qué es el let?

    Se declara de la misma forma que el var. Sin embargo, se usa solo cuando que usar una variable en unicamente una estuctura, bloque o una funcion en particular, declara que solo se puede usar ahi y no en otro lugar.

    Ejemplo

    
                let contador = 0;
                contador = 1; // ✅ Funciona bien
    
                contador = 2;  // ✅ Sigue funcionando
    
                let edad = 20;
    
                edad = edad + 1;  // ✅ Perfecto para ir cambiando valores

  • ¿Qué es const?

    Es una variable que a lo igual que el let, se puede ocupar solo en un lugar en particular, lo unico diferente es que tiene un valor constante... es decir, que no puede cambiar. Si le vamos un valor a alguien, por ejemplo Lucia, Lucia no podrá dejar de llamarse Lucia, jamás.

    Ejemplo 

    
                const nombre = "María";
    
                const edad = 25;
    
                const PI = 3.14159;

    
                const precio = 100;
    
                precio = 200; // ❌ Error!
