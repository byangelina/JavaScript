Variables

  • ¿Qué es var?

    Es un tipo de variable que no lleva tipo de dato, se escribe var nombreVariable;

    Ejemplo 1 
    
const nombreUsuario = "Carlos";  // No va a cambiar
let puntos = 0;  // Va a ir aumentando en el juego

puntos = puntos + 10;  // ✅ Perfecto con let

    Ejemplo 2 
    
if (true) {
var nombre = "Suscribete";

console.log(nombre);
}

    Está bien ✅. Al ejecutar el código en la consola, se imprime Suscribete.


  • ¿Qué es let?

    Se declara de la misma forma que var. Sin embargo, funciona solo cuando que usa una variable en unicamente una estuctura, bloque o funcion en particular. Esto declara que solo se puede usar ahí y en ningún otro lugar.

    Ejemplo 1 
    
let contador = 0;
contador = 1; // ✅ Funciona bien
contador = 2;  // ✅ Sigue funcionando

let edad = 20;
edad = edad + 1; // ✅ Perfecto para ir cambiando valores

    Ejemplo 2 
    
if (true){
let nombre = "Suscribete";
}

console.log(nombre);

    Esto no se puede hacer, da error ❌. En este caso, no existe nombre afuera, este se encuentra dentro del bloque if. Esto ocurre porque es un let, un tipo de dato que solo funciona en un lugar especifico, no en todas partes.


  • ¿Qué es const?

    Es una variable que a lo igual que let, se puede ocupar en un solo lugar en particular, lo unico diferente es que tiene un valor constante... es decir, no puede cambiar. Si le damos un valor a alguien, por ejemplo a Lucia, Lucia no podrá dejar de llamarse Lucia, jamás.

    Ejemplo 1 
    
const nombre = "María";
const edad = 25;
const PI = 3.14159;
const precio = 100;

precio = 200; // ❌ Error!
    Ejemplo 2 
    
const email = "nombre@mail.com";
email = "otro_mail@mail.com"

    Esto no se puede hacer ❌ porque email ya tiene un nombre asignado. A email no le puedo dar otro valor, porque const es único.
